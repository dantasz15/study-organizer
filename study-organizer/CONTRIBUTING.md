# Contribuindo para o Study Organizer 📚

Obrigado por se interessar em contribuir para este projeto! O **Study Organizer** é uma ferramenta focada em ajudar estudantes neurodivergentes a organizar suas rotinas de estudo.

## 🚀 Como Contribuir

1.  **Faça um Fork** do repositório.
2.  **Crie uma Branch** para sua funcionalidade ou correção:
    ```bash
    git checkout -b feature/nova-funcionalidade
    ```
3.  **Desenvolva e Teste**: Certifique-se de que todos os testes estão passando:
    ```bash
    npm test
    ```
4.  **Verifique o Lint**: Garanta que o código segue os padrões do projeto:
    ```bash
    npm run lint
    ```
5.  **Faça o Commit** das suas alterações seguindo o padrão de mensagens claras.
6.  **Envie o Push** para sua branch:
    ```bash
    git push origin feature/nova-funcionalidade
    ```
7.  **Abra um Pull Request** detalhando as mudanças realizadas.

## 🛠️ Padrões de Código

-   Siga as regras definidas no `.eslintrc.js`.
-   Adicione testes unitários para novas funcionalidades.
-   Mantenha a documentação atualizada no `README.md` se necessário.

## 📝 Código de Conduta

Seja respeitoso e colaborativo. Este projeto visa a inclusão e o apoio a estudantes com diferentes perfis cognitivos.
